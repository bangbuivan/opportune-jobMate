1.3 Mục tiêu dự án
CareerForge AI là bộ công cụ hỗ trợ nghề nghiệp tích hợp AI, được xây dựng trên nền tảng Streamlit và FastAPI, cho phép người dùng:
Phân tích và tối ưu hóa CV theo chuẩn ATS (13 tiêu chí kiểm tra) kết hợp đánh giá AI từ Gemini
So khớp CV với mô tả công việc (JD) để tính điểm tương thích, xác định kỹ năng còn thiếu và đề xuất tài nguyên học tập
Gợi ý vị trí công việc phù hợp nhất dựa trên phân tích kỹ năng và so khớp ngữ nghĩa FAISS (CareerMatch)
Tạo CV chuyên nghiệp với biểu mẫu động, nhiều theme (Modern, Minimal, Harvard, Standard), và tính năng nâng cấp nội dung bằng Gemini AI
Luyện phỏng vấn thử (Mock Interview) với câu hỏi tùy chỉnh theo CV và JD, kèm đánh giá chi tiết và chấm điểm từ AI

5. QUY TRÌNH LÀM VIỆC
5.1 Lịch báo cáo
Báo cáo định kỳ: Hàng tuần gửi báo cáo tiến độ cho GVHD qua email/Google Meet
Nội dung báo cáo: Review công việc tuần trước → Cập nhật tiến độ → Kế hoạch tuần mới → Ghi chú vấn đề gặp phải
Liên hệ GVHD đột xuất khi có blocker kỹ thuật cần hỗ trợ

5.2 GitHub Workflow
Branch strategy: main (production) ← dev (integration) ← feature/[tên-tính-năng]
Quy tắc commit message: feat: / fix: / refactor: / docs: / test: + mô tả ngắn tiếng Anh
Pull Request: tự review code trước khi merge vào dev
Code freeze: Tuần 7 – chỉ merge hotfix, không thêm feature mới

5.3 Giao tiếp & Báo cáo
Công cụ: GitHub Issues (theo dõi task + bug) + Google Drive (tài liệu)
Cập nhật tiến độ lên GitHub Issues vào cuối mỗi ngày làm việc
Tổng hợp báo cáo tuần gửi GVHD trước buổi họp
File tài liệu lưu trên Google Drive cá nhân, chia sẻ với GVHD

6. DANH SÁCH TÀI NGUYÊN
6.1 API & Dịch vụ bên ngoài
Google Gemini API – Phân tích CV bằng AI, viết Cover Letter, nâng cấp nội dung CV, chấm điểm phỏng vấn
LinkedIn / Indeed / Naukri – Tổng hợp link tìm kiếm việc làm từ nhiều nền tảng (JobRadar)
Kaggle Open Datasets – Dữ liệu kỹ năng, vai trò công việc IT, danh sách trường đại học

6.2 Thư viện & Framework chính
Streamlit – Framework UI chính cho toàn bộ giao diện multi-page app
spaCy (en_core_web_sm) – Trích xuất thực thể, phân tích cú pháp, tokenization cho CV và JD
FAISS + SentenceTransformers – Lập chỉ mục vector ngữ nghĩa và so khớp vai trò công việc (CareerMatch)
python-docx + Jinja2 – Tạo và xuất CV sang định dạng DOCX/HTML với template engine
RapidFuzz – Fuzzy matching để chuẩn hóa kỹ năng từ CV với từ điển skills.json

6.3 Công cụ phát triển
VS Code + GitHub Copilot – IDE và AI coding assistant
PyMuPDF (fitz) – Trích xuất văn bản từ file PDF resume
Postman – Test API endpoints (FastAPI backend)
Altair + Pandas – Trực quan hóa dữ liệu thị trường lao động trên trang chủ
Google Chrome DevTools – Debug giao diện Streamlit
